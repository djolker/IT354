Week 5 Topics

Objects

Objects in JavaScript are any data type that can be instantiated. This includes basic data types like strings and integers, along with scripted class data types.

Objects can be instantiated using the "new" keyword. 

var instantiatedString = new String("Hello World");

console.log(instantiatedString);

var instantiatedInt = new Number(4);

console.log(instantiatedInt);

Objects can be accessed and set in a number of ways. Below is an example of two concepts, literal notation and bracket notation. The variable 'dog' is being instantiated using literal notation, setting the type, age and name in the first line. It's being accessed multiple times to build the string using bracket notation, where the feature name is given as the key

var dog = {type: "Beagle", age: 5, name:"Sophie"};

console.log(dog['name'] + " is a " + dog['age'] + " year old " + dog['type']);

The instatiation of an object can be apart of a function as well by creating an object constructor. This allows for logic to be implemented into the instantiation process

function dog(type, age, name) {
  this.type = type;
  this.age = age;
  this.name = name;
}

var a = new dog("Beagle", 5, "Sophie");

console.log(a['name'] + " is a " + a['age'] + " year old " + a['type']);

Objects can also be created as a part of a class. This is similar to just using a constructor, except more dynamic features like methods are capable of being used

class Dog {
  constructor(type, age, name) {
    this.type = type;
    this.age = age;
    this.name = name;
  }

  bark() {
    console.log("Bark!");
  }
}

a = new Dog("Beagle", 5, "Sophie");

console.log(a['name'] + " is a " + a['age'] + " year old " + a['type']);

a.bark();
