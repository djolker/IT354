Values and Types

Variables are a basic part of any programming language. In JavaScript, variables have both a VALUE and a TYPE. The value is the data stored in the variable, while the type helps define how to use said data. 

  var stringVariable = "Hello World";
  var intVariable = 4;
  var boolVariable = false;
  
  var stringArray = stringVariable.split(" ");
  
  console.log(stringArray);
  console.log(++intVariable);
  console.log(!boolVariable);


In the above snippit, we can see that variables of three types are instantiated. In lines 7-9, the arrays are being used in different ways, as they are of different variable types.
