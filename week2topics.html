Week 2 Topics

Values and Types

Variables are a basic part of any programming language. In JavaScript, variables have both a VALUE and a TYPE. The value is the data stored in the variable, while the type helps define how to use said data.

  var stringVariable = "Hello World";
  var intVariable = 4;
  var boolVariable = false;
  
  var stringArray = stringVariable.split(" ");
  
  console.log(stringArray);
  console.log(++intVariable);
  console.log(!boolVariable);
Unary Operators

Unary Operators are operations used with only one operand (object be operated on). They're used as quick ways to affect a variable while keeping the code uncluttered and easy to read.

  var x = 1;
  x++;
  console.log(x);
  
  x--;
  console.log(x);
  
  var y = false;
  console.log(!y);
Hoisting

Traditionally, variables are instatiated in the order that the script defines them by reading and executing code from top to bottom. However, JavaScript uses a process called "Hoisting" to instantiate all Variables first, then executing any other code. This allows for code to use variables that have not been defined yet.

  x = 20;
  console.log(x);
  x++;
  console.log(x);
