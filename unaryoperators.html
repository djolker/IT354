Unary Operators

Unary Operators are operations used with only one operand (object be operated on). They're used as quick ways to affect a variable while keeping the code uncluttered and easy to read.

  var x = 1;
  x++;
  console.log(x);
  
  x--;
  console.log(x);
  
  var y = false;
  console.log(!y);

Above, the variable *i* will return 1, as it's been affected by an unary operator *++*.
